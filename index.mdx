## Get started

<Steps>
  <Step title="Install wecom-cli">
    Install the CLI globally via npm — it automatically downloads the correct binary for your platform.

    ```bash
    npm install -g @wecom/cli
    ```
  </Step>
  <Step title="Install the AI Skills (optional)">
    If you plan to use wecom-cli with AI agents, install the bundled skills:

    ```bash
    npx skills add WeComTeam/wecom-cli -y -g
    ```
  </Step>
  <Step title="Configure your credentials">
    Run the interactive setup. You only need to do this once.

    ```bash
    wecom-cli init
    ```

    Choose QR code login (recommended) or enter your Bot ID and Secret manually.
  </Step>
  <Step title="Make your first call">
    Fetch the members visible to your account:

    ```bash
    wecom-cli contact get_userlist '{}'
    ```
  </Step>
</Steps>

<Note>
  wecom-cli currently requires a WeCom account in an organization with **10 or fewer members**. Larger organizations are not yet supported.
</Note>
